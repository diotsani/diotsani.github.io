Go4 Trivia

This project is a mobile trivia game inspired by the engaging quiz experience of games like Trivia Star.

The game was designed to deliver a fast-paced and rewarding question-and-answer experience with a colorful visual presentation, intuitive UI flow, and accessible gameplay for casual players.

Players answer multiple-choice questions across various categories while racing against a countdown timer. The gameplay loop focuses on maintaining engagement through quick interactions, progression pacing, score rewards, and smooth user feedback.

This project is currently still in development and is owned by the studio where I work. During production, I collaborated closely with the development team across multiple gameplay and technical areas to help bring the arcade experience into a modern 3D format.

  • Designing a clean mobile UI that remains readable during fast gameplay sessions.
  • Creating responsive answer interactions with satisfying visual feedback.
  • Maintaining player engagement using timers, progression indicators, and reward systems.
  • Ensuring smooth transitions and lightweight performance on mobile devices.

  • Developed a clean and colorful UI layout optimized for mobile readability and touch interaction.
  • Implemented timed multiple-choice gameplay to create tension and maintain player focus.
  • Added progression indicators and reward feedback systems to encourage continued play.
  • Applied casual-friendly visual styling inspired by modern mobile trivia games while keeping the interface lightweight and easy to navigate.

Key Features

  • Fast-Paced Trivia Gameplay
  • Responsive Mobile UI
  • Progression & Reward System
  • Visual Feedback & Animations
  • Casual-Friendly Experience
